# 项目路线图 (Roadmap)

## 阶段一：基础设施搭建 (Current)

- [x] 初始化项目结构 (uv, src layout)。
- [x] 配置 CI/CD (GitHub Actions)。
- [x] 搭建基础数据库 (SQLite + SQLModel)。
- [x] 实现基础数据录入 API。

## 阶段二：数据积累与清洗

- [ ] **爬虫对接**: 开发或对接爬虫，自动抓取 VTB 直播弹幕和对话。
- [ ] **数据清洗**:
  - [ ] 去除无关噪音 (表情包、重复刷屏)。
  - [ ] 格式化对话对 (Q&A 格式)。
- [ ] **数据管理后台**: 开发简单的 Web 界面查看和标注数据。

## 阶段三：AI 模型微调 (Fine-tuning)

- [ ] **数据集构建**: 将清洗后的数据转换为模型训练格式 (如 JSONL)。
- [ ] **模型选择**: 评估适合的基座模型 (如 LLaMA, Qwen 等)。
- [ ] **训练流程**:
  - [ ] 搭建训练脚本 (LoRA/QLoRA)。
  - [ ] 在本地或云端进行微调实验。
- [ ] **模型评估**: 测试微调后模型的对话风格是否接近目标 VTB。

## 阶段四：应用集成

- [ ] **推理服务**: 将微调后的模型封装为 API。
- [ ] **直播集成**: 实现自动回复弹幕功能。
- [ ] **语音合成 (TTS)**: 集成 VTB 声音克隆。
